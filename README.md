# กาญนะจ๊ะบุรีทริป PWA v1.9.0 — Real Login + GPS Check-in

เวอร์ชันนี้แก้ระบบบัญชีให้เหมือนแอพจริงขึ้น และเพิ่มเช็กอินพร้อมตำแหน่ง GPS ใน Feed

## สิ่งที่เปลี่ยนใน v1.9.0

### Login / Account
- เปิดแอพแล้วต้องเข้าสู่ระบบก่อนใช้งาน
- Admin เริ่มต้นใช้ username `admin` และ password `1234`
- หลังเข้า Admin แล้วสามารถเปลี่ยนรหัสผ่านในหน้า “บัญชีของฉัน” ได้
- Member ใช้ username + password ในการเข้าใช้งาน
- ถ้า username ของ Member ยังไม่มี ระบบจะสร้างบัญชีใหม่ให้ทันที
- รีเฟรชแอพแล้ว session ยังอยู่ ไม่ต้องกรอกใหม่ทุกครั้ง
- เพิ่มปุ่มออกจากระบบ
- ซ่อนคำอธิบายเชิงเทคนิคในหน้าที่ผู้ใช้ทั่วไปเห็น

### Feed / Moment Composer
- ปรับหน้าเพิ่มโมเมนต์ให้ดูเหมือนแอพจริงขึ้น
- เพิ่มปุ่ม “เพิ่ม GPS” สำหรับแนบตำแหน่งปัจจุบันในโพสต์
- โพสต์ที่มี GPS จะแสดงปุ่มดูตำแหน่งเช็กอิน
- รูป/วิดีโอยังคงเก็บแบบเต็มความละเอียด ไม่บีบอัด ไม่ resize

### UI
- เพิ่มหน้า Login แบบ app onboarding
- ปรับข้อความใน Feed ให้เป็นภาษาผู้ใช้ทั่วไปมากขึ้น
- ลดข้อความ Firebase/Drive เชิงเทคนิคในหน้า Feed และ Composer

## วิธีใช้งานเร็ว

1. อัปโหลดไฟล์ทั้งหมดขึ้น GitHub Pages
2. เปิดแอพ
3. เข้าด้วย Admin: `admin` / `1234`
4. ไปที่บัญชีของฉันเพื่อเปลี่ยนรหัสผ่าน Admin
5. ให้เพื่อนสร้าง Member account ด้วย username/password ของตัวเอง
6. Admin ตั้งค่า Firebase และ Drive ใน More เมื่อต้องการใช้ร่วมกันหลายเครื่อง

## หมายเหตุเรื่องความปลอดภัย

แอพนี้เป็น Static PWA ที่รันบน GitHub Pages จึงเป็นระบบ login ระดับแอพสำหรับทริปส่วนตัว ไม่ใช่ระบบยืนยันตัวตนระดับธนาคารหรือระบบสมาชิกเชิงพาณิชย์ หากต้องการความปลอดภัยสูงขึ้นควรต่อ Firebase Auth แบบ Email/Password หรือใช้ backend เพิ่ม

## การอัปเดตจาก v1.8.0

อัปโหลดไฟล์ทั้งหมดทับของเดิม แล้วเปิดแอพใหม่ หากยังเห็นเวอร์ชันเก่าให้ล้าง site data/cache ของเว็บนี้หนึ่งครั้ง เพราะ PWA มี Service Worker cache
